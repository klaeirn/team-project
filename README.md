# Team Project

Welcome to our Quiz App! In this app, you are able to:

- Create your own quizzes
- Take random autogenerated quizzes
- Share quizzes for your friends to take, and take quizzes your friends shared
- See leaderboards among friends to see who's performing the best
- And much more!


In this project, we use the Open Trivia Database Api. The link can be found here:
http://opentdb.com/api_config.php
